About This Page

This webpage is an interactive data visualization using d3. The data is gathered from an online json source (https://raw.githubusercontent.com/prust/wikipedia-movie-data/master/movies.json) that has information about movies, their release year, cast, and genres from 1900 to 2023.

These visualizations include:
  • Total number of movies released each year
  • Total number of movies each year, by genre in the top 5 genres for that time span
  • The top 10 count of actors that were casted in movies for a specified genre
  • The top 10 count of genres for movies that a specified actor was casted in
Instructions:
  • To change the year for the movie line charts, edit the start and end years (the left and right year inputs, respectively), then click the update button. To reset to the original years, click the reset button.
  • To view the top 10 actors for a specific genre, use the drop down select to choose one of the genres in the dataset. After selecting one the bar chart will populate with the corresponding data showing the top 10 counts of actors appearing in that genre.
  • To view the top 10 genres for an actor use the drop down select to select one of the actors in the list. Note that this list only inlcudes the actors within the top 400 count of movie appearances. After selecting an actor the bubble chart will fill with the corresponding data showing that slected actor's top 10 genres.
